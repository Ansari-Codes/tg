The dashboard is where you got about all you ownerships. It has three tabs:
- Dashboard
- Projects
- Settings(Planned)

### Dashboard
Here, you can see your project's data cards and short summary stats. Plus you can also get list of your ongoing projects, last 5. With a likes-per-project graph.

### Projects
Toolbar that contains search input, plus button to create the project and refresh button to refresh project's list. Below it, projects are listed showing thumbnails, likes and views. Pagination options are also available.

### Settings
Planned, will be in feature versions soon!
