This is the page where you create you drawings and projects. In the menu bar, you will see some buttons:
- File: Opens file menu
- - Save: Saves the project
- - Rename: Renames the project
- - Export Canvas: Export canvas as image
- - Revert/Publish: Button to revert or publish the drawing.
- Clear Canvas Button: Clears the drawings on the canvas forcibely
- Run Button: Runs the code
- Stop Button: Stops or terminates the code

Then, there will be a code editor, below is the log area and at right, there is Canvas where drawings will be appeared!
